## Getting Started

1. Go to [Remix](https://remix.ethereum.org/)
2. Paste the code from `SimpleStorage.sol`, `StorageFactory.sol`, and `ExtraStorage.sol` into a new file in Remix
3. Hit `Compile`
4. Hit `Deploy` for whatever contract you want.
